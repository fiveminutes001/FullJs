To update local repo (fetch + merge)

git pull https://github.com/fiveminutes001/Portfolio.git

To update remote repo

git add -A

git diff --check

git commit -m "initial commit 2"

git push origin master

To add longer commit message (-v with reminder of what has changed)

git commit -v

To get current work from remote

git fetch origin

To merge work

git merge origin/master

To find out which part of fetched work needs to be merged

git log --no-merges issue54..origin/master

To show project's roadmap

git log --graph

To view changes

gitk

To show diffrences of commit from HEAD

git diff hash_of_commit

To work correctly with merge and rebase

Rebase local changes before pushing to clean up your work, but never rebase anything that you’ve pushed somewhere.

To initiate a repo

git init

To clone a repo

git clone https://github.com/fiveminutes001/Portfolio.git

To add a local source to remote repo

git remote add origin https://github.com/fiveminutes001/Portfolio.git/

To find out what's the origin

git remote -v

To get rid of pull alert

git config pull.ff only

To show remote details

git remote show origin

To show one line commit log

git log --pretty=oneline

To show commit log

git log

To tag a commit (5 first figures of signature from git log --pretty=online)

git tag -a v1.0 60a85

To show tags

git tag

To show tag details

git show v1.0

To push tags to remote

git push origin --tags

To create a zipped repository copy

git archive --format zip --output 1.zip master

To create a summary of logs by author form version v1.0

git shortlog --no-merges master --not v1.0

To shorten git commands (pull and push)

$ git remote add progit https://github.com/progit/progit2.git (1)
$ git fetch progit (2)
$ git branch --set-upstream-to=progit/master master (3)
$ git config --local remote.pushDefault origin (4)
Add the source repository and give it a name. Here, I have chosen to call it progit.
Get a reference on progit’s branches, in particular master.
Set your master branch to fetch from the progit remote.
