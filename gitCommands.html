To update local repo

git pull https://github.com/fiveminutes001/Portfolio.git

To update remote repo

git add -A

git commit -m "initial commit 2"

git push origin master

To initiate a repo

git init

To clone a repo

git clone https://github.com/fiveminutes001/Portfolio.git

To add a local source to remote repo

git remote add origin https://github.com/fiveminutes001/Portfolio.git/

To find out what's the origin

git remote -v

To get rid of pull alert

git config pull.ff only

To show remote details

git remote show origin
